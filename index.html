QR-API by sarriaroman

This is a free APi to create dinamically QR Codes. Support several types of images like PNG, SVG, EPS.

Install

Just call to this URL: http://new.mobbex.com/api/v1/qr/{IMAGE_TYPE}/{RESPONSE} GET Parameters: IMAGE_TYPE: - JPG - PNG - PDF - EPS - SVG Response: - show ( for HTML IMG label ). - download. POST Parameter: - text ( Text to create the content of the QR Code. Could be a simple text or an URI ).
